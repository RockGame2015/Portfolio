Falling Ball Physics Engine

Interactive Python simulation measuring free-fall motion and velocity of a ball under gravity.

Overview

This project simulates a ball falling under gravity using Python and Pygame. Users can set the initial height, watch the ball drop, and view measurements of the last 5 drops, including time and velocity. It's designed to help understand basic mechanics and physics calculations interactively.

Features

  • Set initial height and start simulation dynamically.
  • Displays last 5 drops with time and velocity.
  • Shows average time and velocity of recent drops.
  • Graphical visualization of height scale and ground reference.
  • Interactive controls: ENTER to start, R to reset, numeric input for height.

Implementation

The engine is built with Python and Pygame. It uses a simple physics formula for free-fall motion:

velocity += gravity * dt
    height += velocity * dt

The positions are converted from meters to pixels for display, and the program keeps a running history of recent drops for measurement display. The simulation runs at 60 FPS using pygame.time.Clock().

Key features implemented:

  • Dynamic input handling and simulation control.
  • Scaling real-world measurements to screen pixels.
  • History tracking for last 5 drops and averages.
  • Simple GUI layout using Pygame drawing functions.
